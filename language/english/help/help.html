Help: XoopSecure

Description

XoopSecure is a module which from admin side helps you to keep your installations clear from malware, virus and helps you to check for other security related items.

Install/uninstall

No special measures necessary, follow the standard installation process â€“ extract the /xoopsecure folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Operating instructions

This module and its operations are very simple:

  • Configure your preferences (e.g. dateformat
  • Add the clean installations filestructure to database
  • Scan for changes since last scan
  • Review files with issues
